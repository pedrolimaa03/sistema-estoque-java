# 📦 Sistema de Estoque em Java

## 📌 Descrição
Sistema de controle de estoque desenvolvido em Java, permitindo cadastrar, atualizar, remover e visualizar múltiplos produtos.

## 🚀 Funcionalidades
- Cadastrar vários produtos
- Remover produtos
- Atualizar quantidade
- Listar todos os produtos

## 🛠️ Tecnologias
- Java

## 📚 Aprendizados
- Estrutura de dados (ArrayList)
- Laços de repetição
- Métodos
- Validação de dados
- Lógica de programação aplicada

## ▶️ Como executar
Compile: 
javac SistemaEstoque.java

Execute: 
java SistemaEstoque.java
